In sectiunea aceasta veti gasi kit-uri, compilatoare, ide-uri, tot ce aveti nevoie pentru a te pune pe picioare

IDE-uri

  1. Visual Studio Link

    Visual Studio este ide-ul oferit de Microsoft. Versiunea aceasta este cea gratuita. Exista 3 tipuri de descarcari :

    • Visual Basic
    • C++
    • C#

  2. Netbeans IDE Link

    Un IDE multifunctional. In primul rand, este multi-platform, ceea ce inseamna ca il puteti folosi si pe Linux, si pe Windows, si pe Mac OS X. De asemenea, Netbeans cunoaste si PHP, si Java, si multe altele.

  3. X-CODE Link

    X-Code este IDE-ul oferit de Apple utilizatorilor Mac OS X. Ultima versiune functioneaza doar pe 10.6 (Snow Leopard) si contine si SDK pentru iPhone. Distractie placuta cu acest IDE, si mai ales Cocoa.

  4. Geany Link

    Geany este, dupa parerea noastra, cel mai de incredere IDE pentru Linux.
